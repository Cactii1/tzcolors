The reward of all initial accounts go to a specified address.

Auctions

An owner of a color can create a new auction and specify the minimum bid amount and the auction duration.

Each bid additionally extends the auction end time by 5 minutes.

The highest bidder can claim the token and the previous owner can claim the reward.

FAQ

A non-fungible token (NFT) is a special type of cryptographic token which represents something unique; non-fungible tokens are thus not mutually interchangeable. This means each color is represented by one unique token that only exists once. This project was created on a voluntary basis during our free time. We don't have any direct commercial motivation.

The reward of each initial auction go to the tz1.. address.

The rewards of all other auctions go 100% to the owner that made the auction.
